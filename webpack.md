Make good webpack.config.js
Then:

```
npm i --save webpack
npm i --save webpack-cli
npm i --save uglifyjs-webpack-plugin
npm i --save babel 
npm i --save babel-loader
npm install --save core-js@2 OR npm install --save core-js@3
npx webpack
```
